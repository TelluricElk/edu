### Модель SVM (линейный случай)

Метод опорных векторов ищет гиперплоскость, которая разделяет классы с максимальным зазором (margin). Для линейно разделимых данных задача сводится к минимизации:

$$\frac{1}{2} \|w\|^2$$

при ограничениях:

$$y_i (w^T x_i + b) \ge 1, \quad i = 1, \dots, m$$

Где:
- $w$ — вектор нормали к гиперплоскости.
- $b$ — смещение.
- $y_i \in \{-1, +1\}$ — метка класса.
- $x_i$ — признаковое описание объекта.

---

### Двойственная задача

Для решения используется двойственная постановка, которая позволяет применить ядра:

$$\max_\alpha \sum_{i=1}^m \alpha_i - \frac{1}{2} \sum_{i,j} \alpha_i \alpha_j y_i y_j (x_i \cdot x_j)$$

при условиях:

$$\alpha_i \ge 0, \quad \sum_{i=1}^m \alpha_i y_i = 0$$

Опорными векторами являются объекты, для которых $\alpha_i > 0$.

---

### Ядра (kernel trick)

Если данные нелинейно разделимы, можно отобразить их в пространство более высокой размерности с помощью функции ядра $K(x_i, x_j) = \phi(x_i)^T \phi(x_j)$.

Популярные ядра:
- **Линейное:** $K(x_i, x_j) = x_i \cdot x_j$
- **Полиномиальное:** $K(x_i, x_j) = (x_i \cdot x_j + c)^d$
- **Радиальное (RBF):** $K(x_i, x_j) = \exp(-\gamma \|x_i - x_j\|^2)$

Ядра позволяют строить нелинейные разделяющие поверхности, оставаясь в рамках линейной оптимизации.
