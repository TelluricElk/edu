### Модель случайного леса

Случайный лес — это ансамбль деревьев решений, каждое из которых обучается на случайной подвыборке данных и случайном подмножестве признаков.

---

### Бутстрап-выборка

Каждое дерево обучается на выборке, полученной **с возвращением** из исходного набора данных (бутстрап). Это создаёт разнообразие между деревьями.

---

### Случайный выбор признаков

При построении каждого узла дерева рассматривается не все $p$ признаков, а случайное подмножество из $k$ признаков (обычно $k \approx \sqrt{p}$ для классификации или $k \approx p/3$ для регрессии). Это уменьшает корреляцию между деревьями.

---

### Прогноз

- Для **классификации** используется голосование большинством деревьев.
- Для **регрессии** — усреднение предсказаний.

---

### Оценка важности признаков

Важность признака вычисляется как среднее уменьшение нечистоты (например, Джини или энтропии) по всем узлам всех деревьев, где этот признак использовался. Это позволяет ранжировать признаки по их вкладу в предсказательную силу модели.
