### Модель логистической регрессии

Логистическая регрессия используется для бинарной классификации. Она оценивает вероятность того, что объект принадлежит к классу 1:

$$p = \frac{1}{1 + e^{-(w^T x + b)}}$$

Здесь:
- $p$ — вероятность принадлежности к классу 1 (значение от 0 до 1).
- $x$ — вектор признаков объекта.
- $w$ — вектор весов.
- $b$ — смещение (интерсепт).
- $e$ — основание натурального логарифма.

Функция $\sigma(z) = \frac{1}{1+e^{-z}}$ называется **сигмоидой** (логистической функцией). Она преобразует линейную комбинацию $w^T x + b$ в вероятность.

---

### Логит-преобразование

Логарифм отношения шансов (логит) выражается как:

$$\log\left(\frac{p}{1-p}\right) = w^T x + b$$

Это позволяет интерпретировать логистическую регрессию как линейную модель в пространстве логарифмических шансов.

---

### Функция потерь (логарифмическое правдоподобие)

Для обучения логистической регрессии максимизируют логарифм правдоподобия (или минимизируют отрицательное логарифмическое правдоподобие):

$$L(w, b) = -\sum_{i=1}^m \left[ y_i \log(p_i) + (1-y_i) \log(1-p_i) \right]$$

Где:
- $m$ — количество объектов в выборке.
- $y_i$ — истинная метка для $i$-го объекта (0 или 1).
- $p_i$ — предсказанная вероятность для $i$-го объекта.

Эта функция штрафует модель сильнее, если она сильно ошибается.

---

### Оптимизация

Минимизация $L(w, b)$ выполняется с помощью **градиентного спуска**. На каждом шаге веса обновляются в направлении, уменьшающем ошибку:

$$w := w - \alpha \frac{\partial L}{\partial w}, \quad b := b - \alpha \frac{\partial L}{\partial b}$$

Где $\alpha$ — скорость обучения.

Производные имеют простой вид:

$$\frac{\partial L}{\partial w} = \sum_{i=1}^m (p_i - y_i) x_i, \quad \frac{\partial L}{\partial b} = \sum_{i=1}^m (p_i - y_i)$$

Это делает градиентный спуск эффективным для больших данных.
