### Модель линейной регрессии

Линейная регрессия описывает зависимость целевой переменной $y$ от признаков $x_1, x_2, \dots, x_n$ с помощью линейной комбинации:

$$y = w_0 + w_1 x_1 + w_2 x_2 + \dots + w_n x_n$$

Здесь:
- $y$ — прогнозируемое значение (целевая переменная).
- $x_i$ — признаки (факторы, влияющие на результат).
- $w_i$ — веса (коэффициенты), определяющие вклад каждого признака.
- $w_0$ — свободный член (интерсепт), базовое значение при нулевых признаках.

---

### Функция потерь (MSE)

Для оценки качества модели используется **среднеквадратичная ошибка** (Mean Squared Error). Она показывает, насколько предсказания модели отличаются от реальных значений:

$$J(w) = \frac{1}{m} \sum_{i=1}^m \left( y_i - \left( w_0 + \sum_{j=1}^n w_j x_{ij} \right) \right)^2$$

Где:
- $m$ — количество объектов в обучающей выборке.
- $y_i$ — реальное значение для $i$-го объекта.
- $x_{ij}$ — значение $j$-го признака для $i$-го объекта.

Цель обучения — найти такие веса $w$, чтобы минимизировать $J(w)$.

---

### Решение методом наименьших квадратов

Для линейной регрессии существует аналитическое решение (закрытая формула):

$$w = (X^T X)^{-1} X^T y$$

Где:
- $X$ — матрица признаков (размер $m \times (n+1)$ с добавленной колонкой единиц для учёта $w_0$).
- $y$ — вектор целевых значений.
- $X^T$ — транспонированная матрица $X$.

Этот метод работает быстро и точно для небольших наборов данных.

---

### Градиентный спуск

Для больших данных используют итеративный метод — **градиентный спуск**. На каждом шаге веса корректируются в сторону уменьшения ошибки:

$$w := w - \alpha \cdot \nabla J(w)$$

Где:
- $\alpha$ — скорость обучения (шаг).
- $\nabla J(w)$ — градиент функции потерь (вектор частных производных).

Градиентный спуск постепенно находит оптимальные веса, двигаясь в направлении наискорейшего убывания ошибки.
